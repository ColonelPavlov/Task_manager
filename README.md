# Планировщик задач (Task Manager) 📋⏱️

**Task Manager** — это мощное и интуитивное веб-приложение для управления личными и командными задачами. Оно помогает организовать рабочий процесс, контролировать дедлайны и визуализировать прогресс с помощью гибких инструментов: от простых списков до канбан-досок и календаря.

## Основные возможности ✨

*   **Управление задачами:** Создавайте задачи с подробным описанием и приоритетами.
*   **Гибкая организация:** Группируйте задачи по проектам и используйте теги для быстрой фильтрации.
*   **Канбан-доска:** Перетаскивайте (Drag-and-Drop) задачи между статусами ("Нужно сделать", "В процессе", "Готово") для наглядного отслеживания прогресса.
*   **Совместная работа (планируется):** Возможность приглашать других пользователей в проекты для командной работы.

## Модели данных (Сущности) 🗂️

Проект построен вокруг трех основных сущностей:

### Проект (Project)
Объединяет задачи в смысловые группы (например, "Разработка сайта", "Личные дела").
*   Название
*   Описание (опционально)
*   Дата создания
*   Участники (для командной работы)

### Задача (Task)
Базовая единица работы.
*   **Название:** Краткое описание того, что нужно сделать.
*   **Описание:** Подробности, чек-листы, комментарии.
*   **Дедлайн (Deadline):** Дата и время, к которым задачу нужно выполнить.
*   **Статус (Status):** `Нужно сделать` / `В процессе` / `Готово` / `Отложено`.
*   **Приоритет (Priority):** `Низкий` / `Средний` / `Высокий` (влияет на цветовую индикацию).
*   **Проект (Project):** Связь с конкретным проектом.
*   **Теги (Tags):** Метки для гибкой категоризации.

### Тег (Tag)
Гибкая метка для маркировки задач по темам, не привязанным к проекту (например: `баг`, `дизайн`, `встреча`).
*   Название
*   Цвет (для визуального отличия)

## Интерфейс и Фичи 🚀

### 🖱️ Drag-and-Drop Канбан
В режиме доски задачи представлены в виде карточек, сгруппированных по колонкам-статусам. Чтобы изменить статус задачи, достаточно просто перетащить карточку из одной колонки в другую. Это делает процесс работы быстрым и наглядным.

## Роли и Доступ (Управление доступом) 🧑‍💻👥

*   **Пользователь:**
    *   Полный доступ ко всем своим задачам, проектам и тегам.
    *   Создание, редактирование и удаление своих сущностей.

*   **Администратор:**
    *   Администрирование, ограничение пользователей, закрытие проекта (проектов)
